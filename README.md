# Vault-Tec - Retexture

<img width="2851" height="1603" alt="Preview" src="https://github.com/user-attachments/assets/0ca4c1b7-5db0-4ce7-a86c-1f0ff76103b0" />

Retexture and cleanup addon for **Vault-Tec - Handy things for your vault!**

This mod updates a large part of the original Vault-Tec furniture set with new textures, adjusted icons, and a few extra building defs and patches to make the set feel more consistent in RimWorld 1.6.

## Requirements

- RimWorld 1.6
- `Twink.Vaultfurniture` / **Vault-Tec - Handy things for your vault!**

This mod is a dependent add-on, not a standalone content pack.

## What This Mod Adds

### New furniture

- Vault folding chair
- Vault work chair
- Vault table (1x2)
- Vault table (2x2)
- Vault table (2x4)
- Vault table (3x3)

### New facility items

- Tool chest
  Adds `+5%` work speed to supported workbenches.
- Diagnostic cart
  Adds `+10%` work table efficiency to supported workbenches.
- Hospital screen
  Adds small bonuses to immunity gain, surgery success, and bed rest effectiveness.
- IV drip stand
  Adds immunity gain support for nearby beds.
- Medical cart
  Adds tend quality and surgery success bonuses for nearby beds.

### Security changes

- Adds a dedicated `ceiling turret gun` def.
- Reworks the Vault-Tec mini turret into a ceiling-mounted style turret with updated base texture, icon, and behavior patching.

## Retexture Coverage

This package includes retextures and icon adjustments for a broad selection of existing Vault-Tec content, including:

- Furniture
  Chairs, tables, wardrobes, lamps, wall lamps, and other interior pieces.
- Medical items
  Divider/screen and related hospital support items.
- Temperature and vent pieces
  Wall-mounted vault vent, heater, cooler, and freezer variants.
- Security content
  Ceiling turret textures and related visuals.
- Production and power visuals
  Updated icons and presentation for several work and power buildings.
- Apparel
  Vault suit, helmet, jacket, and Pip-Boy textures.

## Patch Notes Summary

- Updates many UI icons so build menu previews line up better with the actual sprites.
- Expands facility links for Vault-Tec workbenches and hospital beds.
- Makes the main vault vent wall-mount capable and paintable.
- Adjusts labels, graphics, and icon settings for several inherited Vault-Tec defs.

## Load Order

Load this mod **after** the original Vault-Tec furniture mod.
